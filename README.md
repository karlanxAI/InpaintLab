# Multi Mask Tool

A standalone local inpainting helper for Forge / Forge Neo workflows.

## Download

Go to **Releases** on the right side and download the latest full package.

## Current release

v1.0.0 - First public all-in-one release.

## Install

1. Download the latest `.zip` from Releases.
2. Extract it somewhere safe.
3. Run the included launcher/start file.
4. Follow the setup instructions inside the package.

## Official version

Only download official releases from this repo:
https://github.com/karlanxAI/multi-mask-tool
